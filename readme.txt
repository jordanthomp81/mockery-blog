Kihon - Basic Blogging Theme
==================================
Kihon is a basic theme great for simple blogging. It has a classic design matched with a modern touch. You can also customize the header size, upload your own logo, change colors of the header, footer, theme accent color, menu links and more. Kihon is especially great for starters.
